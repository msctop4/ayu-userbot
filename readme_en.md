
# ZNode UserBot

> 🇷🇺 **Looking for the Russian version?**  
> **[Read the documentation in Russian](https://github.com/msctop4/Znod-userbot/blob/main/readme_ru.md)**

A powerful and feature-rich Telegram UserBot built with performance, simplicity, and reliability in mind.

## ✨ Features

- 🟢 Advanced online status system (Onlineer)
- ⚡ Fast and lightweight
- 🤖 Dozens of built-in UserBot commands
- 🛠️ Easy configuration
- 🔒 Stable and reliable
- 📦 Modular architecture
- 🚀 Designed for daily use

## 📋 Included Commands

ZNode comes with a wide range of built-in commands, including:

- Online status management
- Utility commands
- Account management
- Chat tools
- Message utilities
- Media tools
- Profile management
- And many more...

## 🚀 Installation

```bash
git clone https://github.com/msctop4/Znod-userbot.git
cd Znod-userbot
pip install -r requirements.txt
python main.py
```

## ⚙️ Configuration

Configure the bot by editing the configuration file and adding your Telegram API credentials.

## 📖 About

ZNode is primarily focused on providing one of the best online status systems while also including a collection of essential UserBot features for everyday Telegram usage.

The project is designed to be fast, stable, and easy to extend with new modules.
